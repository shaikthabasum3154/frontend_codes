Hyperlink

  • To link one HTML file with another HTML file programmers using hyperlink concept.
  • To create the hyperlinks in HTML programmers can use 3 ways text content ,buttons ,images , videos.
  • it takes 2 important values
  • href:
  • Used to mention the next HTML file name /URL which has to be displayed after user clicks.
  • target:
  • Target used to open the hyperlink in a new tab of browser and takes value as blank whereas blank is used to open a free hyperlink in new tab.
  • <a href="html filename" target="blank"> new user ?</a>
  • Note:1.Create hyperlink with button make you sure assign button value for type attribute.
  • For hyperlink reference attribute programmers can assign HTML file name /image /video file name.
