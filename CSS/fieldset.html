FIELDSET

  • It is used to get the different formats to the HTML element as a well programmer can take input from user.
  • fieldset is used to represent the truth of different field at one place it is easier to understand.
  • My default field set will extend from starting come off browser to entering corner.
  • In order to reduce dimensions of the field set programmer have to make use of CSS properties like height and width in the style attribute.
  • To measure width and height in the CSS programmers using pixels as measurement unit with representative by..
  • For example:
  • <fieldset style="width:100px; height:20px;"> </fieldset
  • Legend:
  • Legend tag it is used to provide the content on the line of the fieldset.
  • In order to take the data from the user programmers have to make use of input.
  • Input blocks on the browser we have to make use of CSS properties as border = 0px; outline=0px;.
