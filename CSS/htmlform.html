HTML FORMS


1. As a programmer we use d html forms to take values from users into the application.

2. programmatically html forms consists two important components
   * Label
   * Input box

3. Labels are the directions provided for users to enter appropriate data whereas input is the actual datawhich is provided by the user.

4. Labels are indicated by using <label> </label> and to get data from user we have to create input box and it is represented by using <input> </input>

5. to create a html forms we have to use "<form> </form>" tag.

INPUT TYPE and their behaviour

<input type = "text">   ---> user can enter alphabets, numbers, and special symbols.

<input type = "number">   ---> user can enter only numbers.

<input type = "password">   ---> user can enter alphabets, numbers, and special symbols but all will be replaced with solid sphere.

<input type = "Submit">   ---> user can submit the form by form by clicking on it.

<input type = "radio">   ---> user will be given with radio button which results in single selection.

<input type = "checkbox">   ---> user will be given with checkbox which results in multiple selections.

<input type = "date">   ---> user can enter the date in the format dd-mm-yyyy and also get the calendar view.

<input type = "reset">   ---> user can cleae the values which entered by them.

<input type = "month">   ---> user will get the calendar with only month and year.

<input type = "time">   ---> user will get the option to enter the time with different values.

<input type = "email">   --->user can enter a valid pattern of email-id in input box.

<input type = "file">   ---> user will get an option to upload the file.

* By default if programmer doesn't used "type" attribute with < input > tag there then it have default value as "text."

* To change the name of submit button with < input > tag programmer are using "value" attribute.

NOTE

1. As a programmer we can't pass "year" as a values for type attribute.

2. Date value of type attribute will show the format depends on users system.

3. To create watermark on the input box we make use of "placeholder" within <input>.

4. To make fields mandatory on webpage in html forms with input tags make use of "required" attributes.
