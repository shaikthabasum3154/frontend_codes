  • Linear-gradient:
  • Linear gradient will multicolour in a boat vertical format programmer able to change the colour direction by specifying a degree before all colours as well as providing an appearance percentage for each colour.
  • Radial-gradient:
  • To get multi colour in circular format make use of radial gradient where before all the colours the programmer can specify circle to get an exact circular format as well as change the percentage of appearance.
  • Conic-gradient:
  • Get a multicolour in cone format use a conic gradient where the programmer can change the degree for individual colours
