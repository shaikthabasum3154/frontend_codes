Text Formatting tags

  • <em>
  • It stands for emphasis tag.it is used to make the text contents appear in italic format
    • for example:<em> this is italic text</em>
  • <strong>
  • it is used to make the text contents appear in bold format.
    • for example:<strong> this is strong text</strong>
  • <ins>
  • It stands for insertion tag.To give the underline for the text content on the webpage
    • for example:<ins> this is insertion text</ins>
  • <del>
  • It stands for deletion tag.it is used to give the text content strike.
    • for example:<del> this is deletion text</del>
  • <sup>
  • It stands for superscript which is used to provide the cpntents in the above th text
    • for example:<sup> this is superscript </sup>
  • <sub>
  • It stands for subscript .it is used to make the text contents appear in below the text
    • for example:<sub> this is italic text</sub>
