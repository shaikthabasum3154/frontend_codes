Font-Properties

  • Font-weight:
  • Programmer wants to remove bold format from the text use the normal value to make text contacts appear in the board format on the browser use bold value.
  • Font-size:
  • It is used to increase or decrease the size of the text content use the value in the form of pixels.
  • Font-family:
  • It is used to provide different fonts for text contents on the web pagehit accepts following major values that are cursive Kama sansarKama serifKama Georgia monospace, fantasy.
  • Box-shadow
  • This property is used to give the shadow around the box.
  • It works same as text shadow the only difference is text shadow used to give shadow for textcontent and box shadow used to give shadow around boxe/ textbox.
  • Border-radius:
  • It is used to convert sharp edges corners into curved edges for the HTML element.
