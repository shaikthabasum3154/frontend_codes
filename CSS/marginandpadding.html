Margin and Padding Properties

    Margin property:

  • It is used to create spaces around HTML elements.
  • Programmers can apply margin to all sites as well as specific side of the element.
  • To apply margin to the specific side of an element we have to make use of following CSS properties.
  • 1.Margin- left 2.margin- right 3.margin- bottom4. margin- top.
